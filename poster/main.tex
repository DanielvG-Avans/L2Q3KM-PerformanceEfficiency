            \item workloads groeien in volume of complexiteit
            \item lagere client memory en totale cost belangrijk zijn
          }
        \end{column}
        \begin{column}{0.48\linewidth}
          \choicebox{AccentBlue}{Kies CSR als}{
            \item shell-first gedrag zwaarder weegt

            \item je de trade-off op echte devices expliciet hebt gevalideerd
          }

      \end{columns}
